正版验证处于关闭状态，如需开启，请修改服务端根目录下的“server.properties”文件中的“online-mode”为true，可选择删除皮肤插件，也可以将白名单插件移除使用服务器自带的白名单功能
其他配置可直接在游戏中使用help获取，除特殊说明，所有插件均已汉化
控制台中存在报错为正常状态（由于国内网络特殊原因导致插件检查更新失败），若不影响正常游玩忽略即可。
白名单默认处于关闭状态，如需开启请在控制台中执行“easywl on”或在游戏中执行“/easywl on”

白名单用法：（控制台中无需输入斜杠）
1.在游戏或控制台中执行“/easywl add <玩家名>”添加白名单
2.在游戏或控制台中执行“/easywl remove <玩家名>”删除白名单
3.在游戏或控制台中执行“/easywl list”查看白名单列表
4.在游戏或控制台中执行“/easywl on”开启白名单
5.在游戏或控制台中执行“/easywl off”关闭白名单
6.在游戏或控制台中执行“/easywl reload”重新加载
